# OC Remote v1.6.3 — Release Notes

Patch release focused on chat UX polish for review flows.

## Highlights

- Fixed empty chat bubbles shown when launching review from session actions or slash commands.
- Replaced command-only empty bubble state with readable command labels so review actions remain visible and reversible via existing message undo/revert flow.
- Added a leading icon for **Review changes** in the session menu for better action discoverability.
