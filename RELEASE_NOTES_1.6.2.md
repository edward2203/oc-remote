# OC Remote v1.6.2 — Release Notes

Minor release focused on shell command ergonomics, session action discoverability, and cleaner chat/settings controls.

## Highlights

- Improved shell flow for active sessions: shell-mode send is now blocked while the agent is busy, avoiding failed shell submissions during running tasks.
- Restored `/shell` in slash suggestions so new users can quickly discover and enter shell mode without hidden gestures.
- Added **Review changes** to the session menu, mirroring OpenCode `/review` behavior and running review on current uncommitted changes.
- Removed the dedicated shell toggle button from the chat composer and removed its related Settings option to simplify input controls.
- Updated localized strings after English source changes.
